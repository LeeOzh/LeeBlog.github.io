Vue组件化步骤

定义一个组件

  • Vue.extend( )

####注册一个组件（全局/局部）

  • Vue.component( )

  • 组件的命名：

    • 可以是驼峰命名，也可以是-语法

  • 使用组件的名字：

    • 如果是驼峰命名（MyCpn）,可以直接使用驼峰来使用（）,也可以使用-分隔来使用（）。
    • 如果是-语法，只能使用-分隔来使用。

父传子

  • 父组件通过在子组件中定义props向子组件传递数据

    • props的值有两种定义方式：

      • 1.字符串数组，数组中的字符串就是传递时的名称

      • 2.对象，对象可以设置传递时的类型，也可以设置默认值等
